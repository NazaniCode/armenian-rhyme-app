🎓 ALGORITHM OVERVIEW
═══════════════════════════════════════════════════════════════════════════════

HOW IT WORKS:

1. EXTRACT RHYME TAIL
   • Find the last vowel in the word's IPA representation
   • Extract from that vowel to the end
   • This is the "nucleus + coda" (rhyming part)

2. FEATURE-AWARE COMPARISON
   • Compare each phone (sound) with feature awareness
   • Consonants: compared by place and manner
   • Vowels: compared by height and backness
   • Similar phones cost less than different ones

3. LEVENSHTEIN DISTANCE
   • Calculate modified edit distance with feature costs
   • Results in similarity score (0-1)

4. CLASSIFY RHYME TYPE
   • Perfect: identical tails (1.0)
   • Near-perfect: ≥85% similar + same vowel
   • Assonance: same vowel + ≥60% similar
   • Slant: ≥50% similar
   • None: filtered out

5. SORT RESULTS
   • By rhyme type quality
   • Then by similarity score
   • Return top N results
